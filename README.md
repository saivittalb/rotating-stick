# Rotating Stick
üéã An OpenGL framework animation which rotates a stick by taking rotation speed as the input. Available only for macOS users with Xcode.

## Setup
- Clone this repository.
- Download Xcode.
- Navigate to ```Rotating Stick.xcodeproj```
- Then click on the file and open up it up with Xcode.
- Build and run the project.
- Enjoy the code and animation!
